========================================
    MetaCube インストールガイド
========================================

このたびはMetaCubeをご購入いただき、ありがとうございます。
このガイドに従って、簡単に設定を完了できます。

========================================
📋 必要なもの
========================================

1. iPhone 7 または iPhone 8（脱獄済み）
2. AutoTouch（インストール済み）
3. Instagram アプリ（ログイン済み）
4. このスクリプトファイル一式

========================================
📦 ファイル構成
========================================

MetaCube/
├── main.lua           # メインスクリプト
├── config.lua         # 設定ファイル（要編集）
├── functions/         # 機能モジュール
│   ├── timeline.lua   # タイムラインいいね
│   ├── follow.lua     # フォロー機能
│   ├── active.lua     # アクティブユーザー機能
│   └── utils.lua      # ユーティリティ
└── README_JP.txt      # このファイル

========================================
🚀 インストール手順
========================================

ステップ 1: ファイルをAutoTouchにコピー
----------------------------------------
1. iPhoneをPCに接続
2. iFunboxまたはiMazingを開く
3. AutoTouch > Scripts フォルダを開く
4. MetaCubeフォルダごとコピー

または

1. AutoTouchアプリを開く
2. 「+」ボタンをタップ
3. 「Import」を選択
4. ファイルをインポート

ステップ 2: config.lua を編集
----------------------------------------
1. AutoTouchでconfig.luaを開く
2. 以下を編集:

   plan = "basic"    # あなたのプランに変更
   ↓
   plan = "standard" # または "premium"

3. 必要に応じて他の設定も調整
   - daily_limits: 1日の上限
   - intervals: 動作間隔
   - active_hours: 動作時間帯

ステップ 3: 実行
----------------------------------------
1. Instagramアプリを開く
2. ホーム画面（タイムライン）を表示
3. AutoTouchを開く
4. main.luaを選択
5. ▶️ 実行ボタンをタップ

========================================
⚙️ 設定カスタマイズ
========================================

【ベーシックプラン設定例】
daily_limits = {
    likes = 500,      # 1日500いいねまで
    follows = 0,      # フォロー不可
    unfollows = 0,    # アンフォロー不可
}

【スタンダードプラン設定例】
daily_limits = {
    likes = 1000,     # 1日1000いいねまで
    follows = 200,    # 1日200フォローまで
    unfollows = 200,  # 1日200アンフォローまで
}

【プレミアムプラン設定例】
daily_limits = {
    likes = 2000,     # 制限なし（推奨値）
    follows = 500,    # 制限なし（推奨値）
    unfollows = 500,  # 制限なし（推奨値）
}

========================================
🎯 推奨設定
========================================

安全な運用のための推奨値:

1. 新規アカウント（フォロワー < 1000）
   - いいね: 300-500/日
   - フォロー: 50-100/日
   - 間隔: 30-60秒

2. 成長期アカウント（フォロワー 1000-5000）
   - いいね: 500-1000/日
   - フォロー: 100-200/日
   - 間隔: 20-40秒

3. 確立アカウント（フォロワー > 5000）
   - いいね: 1000-2000/日
   - フォロー: 200-300/日
   - 間隔: 15-30秒

========================================
⚠️ 注意事項
========================================

1. 初回は少ない数から始めてください
2. 徐々に数を増やしていきましょう
3. 深夜は避けることを推奨
4. 1日の上限は必ず守ってください
5. エラーが出たら一度停止してください

========================================
🆘 トラブルシューティング
========================================

Q: スクリプトが動かない
A:
- AutoTouchが最新版か確認
- Instagramがログイン済みか確認
- config.luaのプラン設定を確認

Q: すぐに止まってしまう
A:
- active_hoursの設定を確認
- daily_limitsに達していないか確認
- エラーログを確認

Q: いいねができない
A:
- Instagramアプリを再起動
- AutoTouchを再起動
- iPhoneを再起動
